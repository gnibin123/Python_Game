Text-Based Game with Flask
Welcome to the Text-Based Game with Flask! In this game, you'll embark on an adventure through various rooms and interact with the environment using text-based commands.

Getting Started
To play the game, follow these steps:

Prerequisites
Python (3.x) installed on your computer
Flask framework installed (you can install it using pip install Flask)
Installation
Clone or download this repository to your local machine.

bash
Copy code
git clone https://github.com/yourusername/text-based-game.git
Navigate to the project directory.

bash
Copy code
cd text-based-game
Run the Flask application.

bash
Copy code
python app.py
This will start the Flask development server.

Open a web browser and go to http://localhost:5000/ to access the game.

How to Play
You will start in a room with a description of your surroundings.
Use the text input field to enter commands.
Available commands:
look: Examine your current surroundings.
inventory: View your inventory.
go <direction>: Move to another room (e.g., go north).
Explore the rooms, collect items, and solve puzzles to progress in the game.
Pay attention to the descriptions and hints provided to make decisions.
Game Logic
The game logic is defined in the Python code (app.py) and includes the initial game state, room descriptions, exits, and items.
You can customize and extend the game by modifying the Python code and HTML templates located in the templates folder.
Examples
Here are some example commands to get you started:
look: Examine your current surroundings.
inventory: Check your inventory.
go north: Move to a room to the north (if available).
go south: Move to a room to the south (if available).
